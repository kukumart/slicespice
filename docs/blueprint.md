# **App Name**: Slice & Juice

## Core Features:

- About Us Page: Display information about the Slice&Juice restaurant, its history, and mission.
- Menu Display: Showcase the restaurant's menu items with images and descriptions.
- Location Finder: Provide an interactive map showing the restaurant's location.
- Order Form: Enable users to select menu items, customize their order, and view a summary.
- Delivery System: Allow users to provide delivery details and submit their order for delivery.
- Order Tracking: Provide updates to the user regarding the state and estimated arrival time of their order.

## Style Guidelines:

- Primary color: Gold (#FFD700) for a premium and luxurious feel.
- Background color: Dark grey (#222222) providing contrast with white text and gold accents.
- Accent color: White (#FFFFFF) for text and key elements, ensuring readability.
- Body and headline font: 'Poppins', sans-serif, for a modern and easy to read UI.
- Implement a modern 'glassmorphism' card system to present content in a visually appealing way. Use easy-to-navigate UX principles.
- Use minimalist icons with gold accents to complement the color scheme and overall design.
- Subtle transitions and animations to enhance user engagement.
